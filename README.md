# 🔢 Character Counter – Real-Time Text Analyzer

A modern and responsive Character Counter web application built using HTML5, CSS3, and JavaScript. This project provides real-time text analysis, allowing users to track character limits instantly with a dynamic progress indicator and clean UI design.

The purpose of this project is to demonstrate strong fundamentals in DOM manipulation, real-time UI updates, and responsive front-end development using vanilla JavaScript.

📌 Project Overview
Character Counter is a dynamic front-end project that showcases:

• Real-time character counting
• Remaining character limit display
• Interactive progress bar visualization
• Dynamic color changes based on usage
• Smooth animations for better UX
• Clean and modern responsive interface

This project focuses on interactive UI behavior and JavaScript event handling to create a smooth and responsive user experience.

🛠 Technologies Used
• HTML5
• CSS3
• Vanilla JavaScript
• Flexbox
• Responsive Design Principles
• Google Fonts (Inter)

✨ Features
🔹 Text Input Area
• Large responsive textarea
• Clean modern input design
• Maximum character limit enforcement

🔹 Real-Time Character Counter
• Displays total characters typed
• Instantly updates while typing
• Uses JavaScript DOM manipulation

🔹 Remaining Character Display
• Shows how many characters are left
• Changes color when nearing limit

🔹 Progress Bar Indicator
• Visual representation of text usage
• Smooth animated progress bar
• Color changes based on usage level

Example:

• Normal usage → Blue
• Near limit → Orange
• Limit reached → Red

🔹 Limit Reached Animation
• Card shake animation when max limit is reached
• Improves user feedback and interaction

📱 Responsive Design
• Mobile-friendly layout
• Works smoothly on tablets and desktops
• Flexible container layout
• Clean typography and spacing

🧠 Development Process
1. Created semantic HTML structure for accessibility and clarity.
2. Designed a modern UI layout using CSS and Flexbox.
3. Implemented real-time character counting using JavaScript event listeners.
4. Built a dynamic progress bar that updates based on text length.
5. Added color transitions and animation feedback for better UX.
